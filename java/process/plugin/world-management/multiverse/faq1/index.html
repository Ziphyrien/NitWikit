常见问题 1

中文世界名乱码​

不要用中文作为世界的名字，用英文。

你想让插件显示世界名时用中文的话，可以使用 世界别名。

世界别名​

所需插件：

  • Multiverse-Core

  • PlaceHolderAPI

  • 聊天插件

  • TAB 和计分板插件

  • 其他你想展示中文世界名的插件

  • v4：安装 papi 的 Multiverse 扩展

  • v5：无需安装，插件自动挂钩 papi

设置别名​

编辑 plugins/Multiverse-Core/worlds.yml

下方展示部分配置

worlds:
    world:
        ==: MVWorld
        hidden: "false"
        alias: ""
        # 省略部分内容
    world_nether:
        ==: MVWorld
        hidden: "false"
        alias: ""
        # 省略部分内容
    world_the_end:
        ==: MVWorld
        hidden: "false"
        alias: ""
        # 省略部分内容

解释：

  • worlds - 插件检索的 YAML 节点，不用管
  • world - 主世界的默认本名 (可在 server.properties 修改)
  • world_nether - 地狱的默认本名
  • world_the_end - 末地的默认本名
  • alias - 这个世界的别名

我们在 alias: '' 中 '' 填入这个世界的别名

如：

worlds:
    world:
        ==: MVWorld
        hidden: "false"
        alias: "主城"
        # 省略部分内容

然后 /mv reload

使用别名​

将变量写到你想展示世界别名的插件配置里

  • %multiverse_world_alias% - v4 写法
  • %multiverse-core_alias% - v5 写法

当然，这个插件要支持使用 papi 变量

接着 重载那个插件
