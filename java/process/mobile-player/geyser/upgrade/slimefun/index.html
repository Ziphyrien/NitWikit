兼容 Slimefun

危险

想要显示自定义物品纹理，你必须在服务端使用 粘液科技资源包，后才可以正常使用

下载​

第一步，下载 资源包，安装到 服务端/geyser/packs 文件夹

第二步，下载 Mapping，安装到 服务端/geyser/extensions 文件夹

另一个扩展的 Slimefun 资源包 下载后全部拖入 Geyser 文件夹即可使用

重启 服务端即可！
